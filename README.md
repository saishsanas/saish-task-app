# SaishTask App

SaishTask is a full-stack task management application built with Spring Boot and React.  
It helps users create task lists, add tasks, track progress, and manage daily work efficiently.

---

## Tech Stack

### Backend
- Java 21
- Spring Boot
- JPA Hibernate
- PostgreSQL / Docker
- Maven

### Frontend
- React (Vite + TypeScript)
- Axios
- Tailwind CSS
- NextUI Components
